# Student Grade Book CLI

A simple Python command-line program to manage student grades.

## Features

- Add student and grade
- View all students
- Update student grade
- Search for a student
- Calculate class average

## Concepts Used

- Dictionaries
- Loops
- Conditional statements
- User input
- Built-in functions (`sum`, `len`)

## Example

```
1. Add student
2. View students
3. Update grade
4. Find student
5. Show average
6. Exit
```
